**Overview**

This project builds a deep learning model using MobileNetV2 to recognize food items from images (using Food-101 dataset) and estimate their average calorie content.
The goal is to help users track dietary intake and make informed food choices.

**Dataset structure**
dataset/
 ├── train/
 │    ├── pizza/
 │    ├── burger/
 │    └── sushi/
 └── test/
      ├── pizza/
      ├── burger/
      └── sushi/
